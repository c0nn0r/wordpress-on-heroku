Box position ( left, right, top, bottom ),1. Slider action ( hover, click ),1. Open / close time ( in ms )1. Start / open / close opacity of Google Plus Badge box1. Chose 1 of 15 unique icons.**Google Plus Badge** require jQuery 1.4+== Installation == Copy the directory '**Google Plus Badge**' to your WordPress '*/wp-content/plugins*' folder.1. Activate the '*Google Plus Badge*' plugin in your WordPress admin '*Plugins*'2. There is all.== Screenshots ==1. 'Plugin' screenshot. 2.
